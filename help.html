This is here to help you make changes to the website!

First you will want to make a Github account. You will get an email from Github asking you to verify the email address.

After you have made an account and are logged in, you will see the home page. You will then need to ask an admin to invite you to the Thames Hare and Hounds repository. An email will be sent to you from github. Follow the instructions given by the email.

Once you have accepted the invite, you can then edit the repository. Click here to take you to the repository.

This is all the code for the website.

How to a create a new post

Click on to _posts. You will see all the current posts. To create a new post click on the + (pluss) icon above all the files. (create under _posts, importance of the file name format...)

You should come to a page that looks like this. The red boxes is where you must fill in information. The first red box
is where you put the title of the page. You start off with the date like so: 2015-09-23
and followed with the title: 2015-09-23-title
After the title you MUST have .md at the end. Md is short for Markdown and is a easy way to style text on your website.
The next red box is what needs to be at the start of all your posts.
---
layout: post
title: "Title of the page goes here!"
categories: You choose what goes here
---
What comes after this is up to you. Here is a website that simpley explains Markdown. Click Here
